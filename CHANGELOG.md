## Release Type Definitions

### Major Releases (X.0.0)

- Significant game rule changes
- Major mechanic additions or removals
- Breaking changes to card formats
- Community supermajority vote required

### Minor Releases (0.X.0)

- New card sets
- New features and mechanics
- Significant tooling improvements
- Community simple majority vote for game changes

### Patch Releases (0.0.X)

- Bug fixes
- Balance adjustments
- Documentation updates
- Technical improvements
- No community vote required

## Versioning Policy

Forge Realm follows semantic versioning with special considerations for game content:

### Code Versioning

Standard semantic versioning applies to all technical components (game engine, tools, website).

### Content Versioning

- **Major**: Fundamental rule changes affecting existing cards
- **Minor**: New cards, mechanics, or sets that don't break existing content
- **Patch**: Balance changes, errata, and clarifications

### Compatibility Promise

- Patch versions maintain full backward compatibility
- Minor versions maintain deck legality in eternal formats
- Major versions may require deck updates for Standard format
