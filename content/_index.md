{{% blocks/lead color="primary" %}}
Forge Realm is a fully open-source trading card game built for tabletop play. The project focuses on transparent development, community governance, unrestricted print-and-play access, and no artificial scarcity.
{{% /blocks/lead %}}

{{% blocks/section color="dark" type="row" %}}
{{% blocks/feature icon="fa-lightbulb" title="Complete Transparency" %}}
All decisions made publicly with full community visibility.
{{% /blocks/feature %}}

{{% blocks/feature icon="fa-users" title="Democratic Governance" %}}
Community votes drive the direction of game development.
{{% /blocks/feature %}}

{{% blocks/feature icon="fa-download" title="Free Accessibility" %}}
Print-and-play access for everyone, no restrictions.
